Files include the closest overpass point for each set of coordinates over the duration of 1 year (2020).
All datetimes given in UTC.

Step fine/coarse in the filenames indicates the sampling used in the orbital propagation when near/not-near the target coordinates, respectively.

Orbital parameters assumed:
Max view angle = 4.736 deg
Swath width = 100 km
Apogee radius = 610e3 m 
Perigee radius = 608e3 m
Inclination = radians(90.0)
Argument of perigee = radians(90.0)
Right Ascension of Ascending Node (RAAN) = radians(0.0)
True anomaly at epoch = radians(0.0)  
Epoch time (reference time for orbit) = AbsoluteDate(2019, 1, 1, 0, 0, 0.0, utc)
